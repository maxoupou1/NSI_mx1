Comment créer une propriété de collision dans Tiled Map Editor

Suivez ces étapes pour ajouter une propriété de collision à une tuile dans Tiled Map Editor :

Étape 1 : Ouvrir votre Tileset

Dans Tiled, ouvrez votre fichier TMX ou votre Tileset. Si votre carte utilise un tileset externe, assurez-vous d'ouvrir le fichier du tileset lié.

Étape 2 : Sélectionner une tuile

Cliquez sur une tuile dans votre tileset pour la sélectionner. Vous verrez ses propriétés dans le panneau de droite, dans l'onglet Propriétés.

Étape 3 : Ajouter une propriété

  1. Dans l'onglet Propriétés, cliquez sur le bouton + en bas de la liste.
  2. Donnez un nom à la propriété, par exemple collision.
  3. Dans la colonne Type, sélectionnez Booléen.
  4. Dans la colonne Valeur, cochez la case pour définir la valeur sur True.
